# InstaRent

App para buscar piso en alquiler.

## Inicio de la app

Pasos a seguir luego de clonar el repositorio.

1. Instalar las dependencias

   ```bash
   npm install
   ```

2. Arrancar la aplicación

   ```bash
    npx expo start
   ```

3. Seguir los pasos que aparezcan cuando se termine de ejecutar el comando.
